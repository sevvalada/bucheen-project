# Bucheen Arayüz Geliştirme

Bu proje, paylaşılan tasarımın birebir koda dökülmeye çalışılmıştır.

## Teknik Özellikler

* **Teknolojiler:** React ve Tailwind CSS kullanılmıştır.
* **Responsive:** Tasarım mobil, tablet ve masaüstü cihazlara tam uyumlu (responsive) hale getirilmiştir.
* **Yapı:** Vite ile optimize edilmiş, performans odaklı ve bileşen (component) tabanlı geliştirilmiştir.
